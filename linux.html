Linux Tutorial

Introduction

Linux is an operating system/kernel that was created by Linus Torvalds in 1991. He devised the operating system as an improvement of the UNIX OS. However, Torvald never meant to commercialize the product. By using Linux, other programmers designed user friendly operating systems like Ubuntu, Knoppix, Debian and Fedora.

Benefits

  • It's Free!
  • Highly secured due to the open source community continuously improving the security features
  • Completely open-source so anyone with the capable skills can help modify and update the product
  • Offers millions of mostly free programs and applications
  • Highly reliable and widely trusted for server use

Basic Commands

pwd > Informs the user which directory they are currently in by providing the absolute path (path that starts from the root)

ls > Informs the user the files that are in the current directory. Hidden fies can be displayed with the command ls -a

cd > Allows the user to go to a specific directory/folder. Ex: cd Documents will take the user to the Documents folder. cd.. allows the user to go back one folder in the directory

mkdir > Allows the user to create a folder/directory

rmdir > Allows the user to delete a an empty folder/directory

rm > Allows the user to delete a directory with files.

touch > Allows the user to create a new file. Ex: touch notes.txt creates a txt file named notes

man > Allows the user to access the manual for a particualr command and find out more about how to use that command. Ex: man touch gives the user information about the touch command

cp > Allows the user to copy files by taking the location of the file to be copied and the location if where to copy

mv > Allows to move files or rename a file. Ex: mv notes newNotes renames the file notes to newNotes

locate > Allows the user to find a file with a specific name or keywor

echo > Allows the user to put some kind of data in a file. Ex: echo insert text here >> newNotes.txt puts the text "insert text here" in the file newNotes.txt

cat > Allows the user to display the contents of a file

sudo > Allows the user to execute a command with administrative or root privileges.

df > Allows the user to display the available disk space in the different partitions of the system.

tar > Allows the user to compress or uncompress files.

zip, unzip > Allows the user to compress files into a zip archive and to extract files from a zip archive

apt-get > Allows the user to install packages

chmod > Allows the user to change permissions on a specific file and to make it executable
